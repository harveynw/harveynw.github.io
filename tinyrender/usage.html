Usage

Minimal example

Include the necessary header files, initialise tinyrender and start the event loop. 
code/usage_minimal.cpp

Adding Objects

Objects can be added to the scene by the engine interface: 
code/usage_add_object.cpp
Current options are: 
code/usage_object_examples.cpp

Add Color/Textures

Altering the appearance of objects, respecting UV data if applying to an .obj file, is straightforward: 
code/usage_object_appearance.cpp
Objects can also be hidden, i.e. skipped during the render pass: 
code/usage_object_hide.cpp

Building for web

tinyrender is ready to be built using the emscripten SDK, but this will require modifying your event loop to be non-blocking: 
code/usage_emscripten.cpp
