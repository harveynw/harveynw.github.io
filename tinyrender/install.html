Install

tinyrender is configured as a CMake project. You can either download the github repository, place it in your own project's root directory, and then: 
code/install_add_subdirectory.cmake
Or even simpler, use FetchContent: 
code/install_fetch_content.cmake
Finally, link tinyrender with your target: 
code/install_cmake.cmake
